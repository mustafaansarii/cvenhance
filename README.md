# CVEnhance

**CVEnhance** is an AI-powered career platform that helps job seekers build ATS-friendly resumes, prepare for interviews, track job applications, and accelerate their careers through AI-driven tools and learning resources. The platform is built on a scalable microservices architecture with secure authentication, AI-powered resume parsing and analysis, subscription management, a credit-based payment system, Redis-powered caching and rate limiting, and server-side PDF generation.

**Live Demo:** https://cvenhance.in

## Overview

CVEnhance enables users to import existing PDF/DOCX resumes, automatically extract structured information using AI, edit resumes through a live form builder or LaTeX editor, and generate professional ATS-friendly PDFs. Beyond resume building, the platform is evolving into a complete career hub with interview preparation, job tracking, AI career assistance, and curated learning resources.

## Features

| Feature                     | Description                                                                                      |
| --------------------------- | ------------------------------------------------------------------------------------------------ |
| **AI Resume Builder**       | Create ATS-friendly resumes with live editing and professional templates.                        |
| **Resume Import & Parsing** | Import PDF/DOCX resumes and extract structured data using AI.                                    |
| **LaTeX Editor**            | Customize the underlying LaTeX and generate high-quality PDFs.                                   |
| **AI Resume Analysis**      | Receive intelligent suggestions to improve resume quality and ATS score.                         |
| **Professional Templates**  | Multiple recruiter-friendly resume designs.                                                      |
| **Authentication**          | Secure Email OTP, Google, and GitHub login.                                                      |
| **Subscription System**     | Credit-based plans with secure payment integration.                                              |
| **Payments**                | Cashfree integration with HMAC-verified webhooks and idempotent credit allocation.               |
| **Performance**             | Redis caching, distributed rate limiting, and optimized backend services.                        |
| **Microservices**           | Independent services for authentication, resume generation, payments, AI, and notifications.     |
| **Career Platform**         | Interview preparation, job tracking, AI assistance, blogs, and career resources *(in progress).* |
